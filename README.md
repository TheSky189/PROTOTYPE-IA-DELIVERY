**IA Delivery – Prototipo IA (Streamlit + Python)**

### Requisitos
- Python 3.10+

### Instalación
```bash
python -m venv .venv
source .venv/bin/activate   # Windows: .venv\\Scripts\\activate
pip install -r requirements.txt
```

**`requirements.txt` sugerido**
```
streamlit
pandas
numpy
networkx
geopy
```

### Ejecutar
```bash
streamlit run app/main_streamlit.py
```
